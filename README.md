# Quite Note

基于SwiftUI的macOS剪切板历史记录和AI提炼工具

## 功能特性

- 🎨 自定义剪切板+AI闪光主题图标
- 🔧 改进的LucideIcons加载机制
- 📱 优化的状态栏图标显示
- 📋 剪切板历史记录管理
- 🤖 AI内容提炼功能

## 系统要求

- macOS 12.0 或更高版本

## 安装方式

### 方式一：下载DMG安装包
1. 下载 `QuiteNote-1.0.0.dmg`
2. 双击挂载DMG文件
3. 拖拽应用到Applications文件夹
4. 首次运行可能需要在"系统偏好设置 > 安全性与隐私"中允许运行

### 方式二：直接使用应用包
1. 下载 `Quite Note.app`
2. 将其移动到Applications文件夹
3. 双击运行

## 使用说明

1. 应用启动后会出现在状态栏
2. 点击状态栏图标查看剪切板历史
3. 使用AI功能提炼剪切板内容

## 构建说明

本项目使用Swift Package Manager构建，支持GitHub Actions自动构建。
